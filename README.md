# ImageTextAdder

Добавляет текст на изображение по средством REST API. Выгружает получившееся изображение в S3 хранилище.

## Endpoints
- `/api/image/new` - создаёт новое изображение.
- `/api/image/{id}` - выдаёт ссылку на изображение.
